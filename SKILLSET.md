# SKILLSET.md - zkdex skills

This document defines all skill modules included in the `zkdex skills` package.

## Included Skill Modules

- `zk-dex-keygen-python`: Python-based `BabyJubJub` key pair generation module. Uses the verified implementation from the `barryWhiteHat/baby_jubjub_ecc` repository.
- `zk-dex-transfer-python`: Python-based transfer note generation module. Used for asset transfers.
- `zk-dex-mint-python`: Python-based mint note generation module. Used to issue new assets.
- `zk-dex-redeem-python`: Python-based redeem note generation module. Used to redeem assets.
- `zk-dex-glossary`: A glossary module that compiles core terms of the zk-DEx project.

## Installation

1. Clone this repository.
2. Place all skill modules into the `projects/tokamak-zk-dex/skills` directory.

## Usage

Each skill module can be used independently. For usage details, refer to the `SKILL.md` file in each module.
